Winter X Games Aspen 2019 is adding three new event from now

by Jeffrey James December 16, 2018 0 2 Views
Share:

In an effort to increase viewership and highlight trending sports, the Winter X Games is adding three new disciplines to their flagship winter competition.

Snowboard Knuckle Huck, Adaptive Snow Bike-Cross and Para Snow Bike-Cross are slated to be added as competitive events for the 2019 edition of the X-Games in Aspen, Colorado.

For the better part of a decade, the X-Games have featured a variety of adaptive sports. Now two new events will feature adaptive athletes, and a third event focuses on a very specific type of snowboarding.

Possibly the most peculiar new addition to the games is the ???Knuckle Huck??? snowboard division, which allows eight athletes to compete head to head in a jam-style format one upping each others tricks until time expires. Instead of attempting traditional tricks on large jumps, the Knuckle Huck division is about creativity and using the jump landings found in the terrain park.
